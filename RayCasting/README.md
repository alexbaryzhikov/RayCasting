## Controls

| Keys          | Actions                 |
| ------------- | ------------------------|
| W, S, A, D    | movement                |
| Arrow keys    | turn, movement          |
| B             | toggle walls visibility |
| N             | toggle walls collision  |
| M             | toggle map              |
| Shift + M     | toggle minimap          |
| +, -          | zoom map                |
| 0             | reset map zoom          |
| Esc           | release mouse           |
