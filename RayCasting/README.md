## Controls

| Keys          | Actions               |
| ------------- | ----------------------|
| W, S, A, D    | movement              |
| Arrow keys    | turn, movement        |
| N             | toggle wall collision |
| M             | toggle map visibility |
| Shift +, -    | toggle map / minimap  |
| +, -          | zoom map              |
| 0             | reset map zoom        |
