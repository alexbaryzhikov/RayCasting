## Controls

| Keys          | Actions                 |
| ------------- | ------------------------|
| W, S, A, D    | movement                |
| Arrow keys    | movement, map scroll    |
| B             | walls visibility        |
| N             | walls collision         |
| M             | map                     |
| Shift + M     | minimap                 |
| F             | map follow mode         |
| G             | map player presentation |
| +, -          | map zoom                |
| 0             | reset map zoom          |
| Esc           | release mouse           |
